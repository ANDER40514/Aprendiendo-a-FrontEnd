ATAJOS GIT/ GITHUB



    Git/GITHUB es una herramienta para subir tus proyectos a las nube, una excelente plataforma para el trabajo equipo ya que se pueden realizar modificiaciones de manera simultanea, y a su vez los integrantes del proyecto hacer una rama (branch) y trabajar independiente



    Fuente:(https://bluuweb.github.io/desarrollo-web-bluuweb/07-01-git/#crear-repositorio)


    Atajos de Git and Github

    Abrir Terminal

    CTRL + SHIFT + ` (AL LADO DEL 1)

    Clonar repositorio

    (git clone URL nombreCarpeta)

    AÃ±adir los cambios pendientes al repositorio

    (git add . )

    Subir commit al repositorio

    (git commit -m " ")

    Realizar Push

    (git push)

    Ver archivos los cuales no han sido registrados

    (git status -s )

    Ver una lista de los Commit reapzado

    (git log --onelone)

    Para ir a un commit en especifico

    (git chekout [ID])

    Para ir a un commit en especificio y borrar los demas archivos a futuros(destructivo)

    (git reset --hard [ID])

    Para ir a un commit en especificio y dejar los demas archivos sin seguimiento para actualizaciones

    (git reset [ID])

    para restaurar los cambios borrados

    (git reflog)

    Deshace los cambios reapzados por un commit anterior creando un commit completamente
    nuevo, todo esto sin alterar el historial de commits.

    (git revert [ID])

    Verificar la Rama en la que estamos

    (git branch)

    Crear una nueva Rama

    (git branch nombreRama)

    Movernos de Rama

    (git checkout nombreRama)

    Nos muestra visualmente las ramas

    (git log --oneline --graph)

    Podemos unir una rama con la nueva, para eso tenemos que estar en la rama que espera los cambios

    (git merge nombreRama)

    Eliminar una rama

    (git branch -d nombreRama)

    Crear una nueva rama y viajar en ella

    (git checkout -b nuevaRama)

    Verificar en que rama nos encontramos

    (git remote -v)

    Aplicar push con la nueva Rama

    (git push --set-upstream origin nombreRama)

    Ver las carpetas

    Git status -s

    Ver un listado de archivos y carpetas

    Ls

    Visualizar los cambios que se realizaran en el commit

    Git diff

    Ver contenido de un archivo

    Cat (documento.txt)

    Agregar un archivo de GIT

    Git add archivo2.txt

    Eliminar un archivo de GIT

    rm archivo2.txt
