Margin y padding es de los mas necesario a tener en cuenta para una buena ejecucion de una responsibe Web

Comenzare desde lo mas inicial a la hora de realizar nuestro proyecto CSS,

Propiedad Color

Es la propiedad que se encarga de cambiar los colores del elemento

Color {

  • color: red ;
  • color: linear-gradient(red,yellow) ;

Propiedad Margin

El margen (margin) es en realidad el espacio alrededor del elemento, el que se encuentra por fuera del borde de esa caja.

margin {

  • Margin-top: 10px ;
  • Margin-bottom: 10px ;
  • Margin-left: 10px ;
  • Margin-right: 10px ;
  • Maring: 10px 5px 6px 8px ; (En orden de las agujas del reloj)

Propiedad Padding

El estilo padding es el espacio alrededor del contenido del elemento pero dentro de sus bordes.

padding {

  • paddin-top: 10px ;
  • padding-bottom: 10px ;
  • padding-left: 10px ;
  • padding-right: 10px ;
  • padding: 10px 5px 6px 8px ; (En orden de las agujas del reloj)

Propiedad width

Establece el ancho de un elemento, sus valores predeterminados vienen en auto (sus valores puedes ser px,em,rem)

width {

  • width: 10px ;
  • min-width: 10px ;
  • max-width: 10px ;

Propiedad Height

Establece la altura de un elemento sus valores predeterminados vienen en auto (sus valores puedes ser px,em,rem)

Height {

  • height: 10px ;
  • min-height: 10px ;
  • max-height: 10px ;

Estilos de texto

Propiedad con la cual decidimos como se alineara el texto

text-align {

  • text-align: left ; (Alineacion de texto a la izquierda)
  • text-align: center ; (Alineacion de texto en el centro)
  • text-align: right ; (Alineacion de texto a la derecha)
  • text-align: justify ; (Alineacion de texto justify o que se expande)

Decoracion de texto

La propiedad text-decoration nos ayudan a proporcionar un estilo al texto

text-decoration {

  • text-decoration: underline ; (subrayado)
  • text-decoration: overline ; (linea por encima del texto)
  • text-decoration: line-through ; (tachado)
  • text-decoration: none ; (Sin decoracion)

Transformacion de texto

Mediante esta propiedad podemos cambiar como estan establecidas las mayusculas y minusculas de un texto

text-transform {

  • text-transform: uppercase ; (Maysuculas)
  • text-transform: lowercase ; (minusculas)
  • text-transform: capitalize ; (Primera Maysuculas)
  • text-transform: none ; (Sin Transformacion)

Indentacion

Propiedad que nos sirve para identar el texto (espacios en las palabras)

text-indent {

  • text-indent: 60px ; (indentacion de la primera linea de un parrafo)

espaciado en las letras

Con esta propiedad podemos designar el espaciado que tendran las letras en el texto

letter-spacing {

  • letter-spacing: 3px ; (Espacio entre caracteres y admite valores negativo)

Espacio entre lineas

Establecer el espacio entre las lineas de texto

line-height {

  • line-height: 1.5 ;

Direccion del texto

Con esta propiedad seleccionamos el valor de como vendra el texto, por defecto viene de derecha a izquierda, pero podemos cambiar eso

direction {

  • direction: rt1 ; (Escribe de derecha a izquierda)

espacio de letras

Designar el espacio de letras

word-spacing {

  • word-spacing: rt1 ; (Escribe entre palabras del parrafo)

sombra al texto

Con esta propiedad añadimos una sombra y el dicho color de esa sombra

text-shadow {

  • text-shadow: 4px 2px blue ; (agrega una sombra azul al texto en 4px horizontal y 2px en vertical)

Fuentes

Las propiedades de fuentes de CSS definen la familia, el tamaño, el peso (negrita) y las variantes.

En CSS; hay dos tipos de nombres de familia:

familias genéricas: las que tienen una apariencia similar (serif, sans-serif o monospace) familias de fuentes: fuentes específicas (Times New Roman, Georgia, Arial, Verdana, Courier New, Lucida Console, etc.)

font-family {

  • font-family: Arial ; (Tipo de fuente de las palabras en Arial)

Importar Fuentes

Podemos importar fuentes de google Fonts y seguir los siguentes pasos para utilizar una fuente:

  1. Importar fuente y añadirlas al CSS: @import url('https://fonts.googleapis.com/css?family=Roboto');

  2. IMPORTAR FUENTES Y añadirlas al HTML como etiqueta link link href="https://fonts.googleapis.com/css?family=Roboto" rel="stylesheet"

  3. Agregar dicha fuente: font-family: "Roboto", sans-serif; /* utilizará la Roboto si está disponible, y si no el grupo sans-serif por defecto. */

Estilos de texto

Implementar varios tipos de estilos de texto

font=style {

  • font-style: normal ; (Texto normal)
  • font-style: italic ; (Texto en cursiva)
  • font-style: oblique ; (Similar a cursiva, pero menos soportado por navegadores)

Tamaño de texto

Definir diferentes tipos de tamaño de texto, con sus respectivos diferentes unidades de medidas, Por defecto el navegador aplica un tamaño de 16px

font-style {

  • font-size: 16px ; (Tamaño normal)
  • font-size: 1em ; (tamaño en em, que permite al usuario re-dimensionar la fuente desde el menú del navegador. 1em = 16px)
  • font-size: 1rem ; (tamaño en rem, relativos al elemento padre anterior)
  • font-size: 2.5em ; (tamaño en em, que permite al usuario re-dimensionar la fuente desde el menú del navegador. 1em = 16px)
  • font-size: 100% ; (tamaño en porcentaje)
  • font-size: 10vw ; (10% del ancho del viewport o ventana del navegador)
  • font-variant: small-caps ; (las minúsculas se convierten en mayúsculas pequeñas)
  • font-variant: normal ; (texto normal)

Bordes

Hay diferentes tipos de borde que podemos ponerle a nuestros elementos

border-radius {

  • border-color: Establece el color del borde en cualquier formato de color aceptado (#000, rgb(0,0,0), etc.)
  • border-width: Establece el ancho del borde en px, em, etc.

  • border-style: solid (continuo)
  • border-style: dotted (con puntos)
  • border-style: dashed (con rayas)
  • border-style: double (doble borde)
  • border-style: none (Sin borde)

  • border-radius: 15px 50px 30px 5px; (4 valores (superior izquierdo, superior derecho, inferior derecho, inferior izquierdo)
  • border-radius: 15px 50px; (2 valores (superior izquierdo e inferior derecho, superior derecho, e inferior izquierdo)
  • border-radius: 15px; (1 único valor para las 4 esquinas)

  • border: 1px solid #000; (Una forma de estilo rapido para definir los 3 anteriores)

Fondos

Podemos establecer fondos para elementos y darle un bonito diseño a nuestra pagina web

background-color {

  • background-color: red; permite aplicar un color de fondo en cualquier formato de color aceptado (#000, rgb(255,0,0), rgba(255,0,0,0.1), etc.)

  • background-image: url("ruta-hasta-la-imagen/imagen.jpg"); Permite aplicar una imagen de fondo

  • background-repeat: no-repeat; (Indica si debemos repetir el fondo cuando el elemento sea más grande que la imagen. Las opciones más comunes son no-repeat, repeat (por defecto), repeat-x, repeat-y)

  • background-size: 100px 100px; (permite establecer el tamaño de la imagen de fondo, también puede ser en porcentajes u otra medida

  • background-position: 10px 50px; (ermite especificar la posición en la que debe empezar la imagen de fondo en forma de par de valores x e y)

  • background: #00ff00 url("smiley.gif") no-repeat; (Permite establecer varias propiedades de las anteriores de forma simultánea)

Colores

Existen hasta unos 140 colores disponibles de una manera normal y hasta unos 16M en hexadecimales

color {

  • color: Tomato; (Utilizando un color keyword.)

  • color: rgb(255, 99, 71); (Utilizando los valores RGB (red, green, blue entre 0 y 255)

  • color: rgba(255, 99, 71, 0.5); (Como el anterior con una transparencia del 50%.)

  • color: #FF6347; Utilizando el valor HEX (red, green, blue en hexadecimal entre 00 y FF).

  • color: hsl(9, 100%, 64%); /* Utilizando los valores HSL (hue o matiz en la rueda de color de 0 a 360, saturation o saturación de 0 a 100%, lightness o luminosidad de 0 a 100%).

  • color: hsla(9, 100%, 64%, 0.5); (Como el anterior con una transparencia del 50%.)

Uso de initial, inherit e important

Son propiedades que es dan gran importancia y tener en cuenta

h1 {

  • font-size: initial ; (*forzar que todos los h1 utilicen su font-size por defecto
  • color: inherit ; (forzar que todos los h1 dentro de section utilicen el color de fuente declarado en la section)
  • color: red !important ; (forzar que todos los h1 dentro de section tengan un color de fuente concreto, incluso cuando en otros lugares del CSS se haya escrito otra cosa
