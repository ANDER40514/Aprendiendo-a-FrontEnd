ETIQUETA HTML

Estructura basica de una pagina

La mayoria de las etiquetas cuantan con una etiqueta de apertura ( >) y una apertura de cierre (/>)





Etiqueta Head

Esta etiqueta se utiliza para incluir las fuentes externas y los elementos de esta etiqueta no son visibles por el lector


Etiqueta NAV

La etiqueta NAV se utiliza para los menu de navegacion en la parte superior


Etiqueta HTML

Es la primera etiqueta en abrir y la ultima en cerrar, esta etiqueta se basa en decirle a la computadora que es un archivo HTML y que es la raiz del archivo


Etiqueta para encabezados H1-H6

Las etiquetas para encabazados y subtitulos son aquellas etiquetas que van desde el h1 hasta el h6 cambiando su tamaño en grado de relevancia


Etiqueta para Parrafo

Las etiquetas P son utilizadas para los parrafos, cualquier parrafo debe ser con su respectiva etiqueta


ETIQUETA !DOCTYPE

La etiqueta se encarga de decirle a la computadora la version del HTML


Etiqueta Tittle

La etiqueta Tittle es la encarga de mostrar un titulo en la posicion de la pestaña, en la parte superior del navegador


Etiqueta Link

Enlaza archivos CSS


Etiqueta script

Enlaza archivos de JavaScript


Etiqueta Body

Elementos visibles en la pagina


Etiqueta main

Representa el contenido dominante de un documento


Etiqueta sub

Esta etiqueta agrega un subindice 19th


Etiqueta sup

Esta etiqueta agrega un subindice 21th


Etiqueta nobr

Esta etiqueta sirve para impedir un salto de linea


Etiqueta b o strong

Sirve para colocar el texto en negrita


Etiqueta i

Etiqueta que sirve para colocar el texto en cursiva


Etiqueta U

Etiqueta que sirve para subrayar un texto


Etiqueta de anclaje (a)

Etiqueta que sirve para incluir un link, tanto local como global y es una etiqueta de cierre automatico


Etiqueta br

Etiqueta que sirve para establecer un salto de linea y es una etiqueta de cierre automatico


Etiqueta hr

Etiqueta que sirve para agregar una linea horizontal y es una etiqueta de cierre automatico


Etiqueta blockquote

Etiqueta que sirve para agregar un texto en formato de cita


Etiqueta img

Etiqueta que sirve para insertar una imagen y es una etiqueta de cierre automatico


Etiqueta video

Etiqueta que sirve para insertar un video


Etiqueta iframe

Etiqueta que sirve para introducir contenido de otras paginas externas.


Etiqueta audio

Etiqueta que sirve para insertar un audio


Etiqueta UL

Etiqueta que sirve para crear una lista desordenada


Etiqueta ol

Etiqueta que sirve para introducir una lista ordenada


Etiqueta Li

Etiqueta que sirve para definir los arituclos de las listas tanto desordenas como ordenadas


Etiqueta footer

Esta etiqueta define el pie de pagina o la parte final de una pagina web


Etiqueta Table

Etiqueta que crea una tabla


Etiqueta Tr

Etiqueta que crea las filas de la tabla


Etiqueta td

Etiqueta que sirve apra crare las columnas de la tabla


Etiqueta th

Etiqueta que sirve para crear el titulo de cada columna


Etiqueta col

Especifica las propiedas de las columna


Etiqueta form

Crea un formulario y dentro de esta etiqueta van todos los demas input, label y button del formulario


Etiqueta Label

Sirve como titulo de un elemento para una interfaz


Etiqueta input

Crea un campo para introducir datos, existen diferentes atributos de input los cuales son, se asocia a input y a textarea

  • input text: para texto normal
  • input number: Para introducir numeros
  • input email
  • input date: para fechas
  • input color
  • input range
  • input file: Introducir documentos
  • input textarea: para comentarios
  • input password
  • input time
  • input day
  • input week
  • input month
  • input submit
  • input type = "checkbox"
  • input type = "radio"

Etiqueta button

Agrega un boton


Etiqueta legend

Representa un titulo para un conjunto de campos


Etiqueta section

El elemento HTML section representa una sección independiente genérica de un documento, que no tiene un elemento semántico más específico para representarlo. Las secciones siempre deben tener un encabezado, con muy pocas excepciones.


Etiqueta abbr

Esta etiqueta es para referirse a una abreviatura
